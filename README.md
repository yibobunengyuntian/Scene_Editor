## 项目介绍：

在qt中使用opengl， QOpenGLWidget；仿游戏引擎的场景编辑器， 导入外部模型文件， 界面交互， 点击选中模型， 移动、旋转、缩放；材质编辑， 灯光， 摄像机；opengl实战练习。
